# 🐾 Pati Dünyası: Senin Dijital Dostun!

**Pati Dünyası**, evcil hayvan severler için tasarlanmış, içinde mini oyunlar ve gelişim süreçleri barındıran eğlenceli bir sanal bebek (Tamagotchi) uygulamasıdır. Kendi dijital dostunu seç, onu besle, onunla oyunlar oyna ve gelişimine tanıklık et!

## 🎯 Projenin Amacı ve Oyunlaştırma Özellikleri

Bu projenin temel amacı, kullanıcılara sorumluluk bilinci aşılarken eğlenceli bir vakit geçirmelerini sağlamaktır. Uygulama içinde yer alan oyunlaştırma (gamification) mekanikleri şunlardır:

*   **Glitter Puan Sistemi:** Mini oyunları başarıyla tamamlayarak puan kazanın ve bu puanlarla evcil hayvanınızın ihtiyaçlarını karşılayın.
*   **Enerji ve İhtiyaç Yönetimi:** Evcil hayvanınızın yemek, uyku ve oyun ihtiyaçlarını dengede tutmanız gerekir. Enerjisi biten bir hayvan dinlenmeye çekilir!
*   **Hafıza Kartı Oyunu:** Zihninizi tazeleyen ve evcil hayvanınıza puan kazandıran entegre bir mini oyun.
*   **Görsel Gelişim:** Evcil hayvanınızın durumuna göre değişen canlı ve gerçekçi piksel art çizimler.

## ⚙️ Nasıl Çalıştırılır? (Installation & Run)

Projeyi yerel bilgisayarınızda çalıştırmak isterseniz aşağıdaki adımları takip edebilirsiniz:

### 1. Depoyu Klonlayın
```bash
git clone https://github.com/sengka/PatiDunyasiApp.git
cd PatiDunyasiApp
```

### 2. Bağımlılıkları Yükleyin
```bash
npm install
```

### 3. Uygulamayı Başlatın
```bash
npm run dev
```

Uygulama varsayılan olarak şu adreste çalışacaktır:  
http://localhost:3000
